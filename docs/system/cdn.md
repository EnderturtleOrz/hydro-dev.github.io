# 使用内容分发网络

## 资源 CDN

在控制面板中将 `server.cdn` 设置项修改为 CDN 域名。（如 `https://cdn.hydro.ac/`，以 `/` 结尾）。  

## 全站 CDN

如果您预算充足，可以使用全站 CDN。

缓存设置可以参考以下设置

| 类型     | 内容                | 缓存行为 |
| -------- | ------------------- | -------- |
| 全部文件 | 全部文件            | 遵循源站 |
| 文件后缀 | jpg,js,html,png,css | 缓存一天 |

同时设置添加回源HTTP请求头

| 头部参数      | 头部取值                              |
| ------------- | ------------------------------------- |
| X-Forward-For | 参考所使用服务商CDN文档来设置用户来源IP |

且您应当修改系统设置中的 `server.xff` 值为 `X-Forwarded-For`，该设置是为了能够让系统正确获取到用户的IP地址。
